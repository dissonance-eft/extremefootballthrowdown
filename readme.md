# Extreme Football Throwdown

A player-vs-player, American football game if there were no rules.

* [Steam Workshop post](https://steamcommunity.com/sharedfiles/filedetails/?id=104625632)
* [Fretta13 gamemode base](https://github.com/jetboom/fretta13)

## Install

1. Put extremefootballthrowdown and fretta13 folders in to garrysmod/gamemodes folder.
2. Add +gamemode extremefootballthrowdown before the +map declaration in your server startup script.

## Maps

A basic pack of maps is available [here](http://steamcommunity.com/sharedfiles/filedetails/?id=244859331) and many community maps are available on the [Steam workshop](https://steamcommunity.com/workshop/browse/?appid=4000&searchtext=eft_&childpublishedfileid=0&browsesort=textsearch&section=readytouseitems&requiredtags%5B%5D=Map&created_date_range_filter_start=0&created_date_range_filter_end=0&updated_date_range_filter_start=0&updated_date_range_filter_end=0).
